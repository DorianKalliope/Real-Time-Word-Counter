# Word Counter

A simple VS Code extension that shows the number of words in the current file.

- Displays word count in the status bar (bottom right)
- Updates automatically as you type
- Works with any text file

## Usage

Just open any text file and start typing. The word count appears at the bottom of the window.

## Requirements

None – works out of the box.
